Getting Started

1. Add your OpenAI API key in the API Key Setup section

2. Drag and drop a file or click to select one

3. Choose an action based on your file type

4. Click "Process File" to generate AI results

Supported File Types

Text: .txt, .docx, .pdf (text extraction)

Images: .jpg, .png, .gif, .webp

Audio: .mp3, .wav, .m4a (transcription)

Troubleshooting

API errors: Check your API key and billing status

File upload issues: Ensure file size is under 25MB

Processing failures: Try a different action or file format
